- [ ] **Quick sort** -> Best: O(n log n), Avg: O(n log n), Worst: O(n²) 
- [ ] **Bucket sort** -> Best: O(n + k), Avg: O(n + k), Worst: O(n²) 
- [ ] **Heap sort** -> Best: O(n log n), Avg: O(n log n), Worst: O(n log n)
